本目录用于存放贺卡的背景音乐文件。

## 如何添加音乐文件

1. 从免费音乐资源网站下载音频文件（如 FreeSound、Pixabay 等）
2. 将文件重命名为上述列表中的文件名
3. 放入此目录即可

注意：音乐文件应为 MP3 格式，建议时长 30秒 - 2分钟。
